Este es el repositorio del portafolio personal de **Juan Esteban Barrios Portela**, Tecnólogo en Análisis y Desarrollo de Software especializado en Backend (Java, Spring Boot, Python, Django).

El portafolio fue construido con un diseño altamente interactivo, oscuro y espacial para reflejar la complejidad técnica y estructura de un sistema backend, usando **Angular 22** con componentes independientes (Standalone) y animaciones fluidas con **GSAP**.

## Caracteristicas

- **Diseno Espacial y Oscuro**: Interfaz que imita un sistema solar de tecnologias giratorias.
- **Micro-interacciones y Animaciones**: Implementadas usando GSAP (ScrollTrigger) y CSS puro para el movimiento orbital interactivo 3D.
- **Totalmente Responsivo**: Adaptado a moviles, tablets y monitores ultrawide.
- **Soporte de Tema Claro/Oscuro**: Arquitectura completa de colores basada en variables CSS adaptables.
- **Optimizado para SEO**: Etiquetas meta configuradas, estructura semantica HTML5 y rendimiento optimizado.
- **Componentes Modulares**: Arquitectura robusta y limpia con Angular Standalone Components.

## Stack Tecnologico del Portafolio

- **Framework**: Angular 22
- **Lenguaje**: TypeScript
- **Estilos**: CSS3 Puro (Variables, Flexbox, Grid, Animaciones 3D)
- **Componentes UI**: PrimeNG 21
- **Motor de Animaciones**: GSAP
- **Iconos**: FontAwesome & Devicons

## Instalacion y Ejecucion Local

Para correr este proyecto en tu entorno local, asegurate de tener instalado [Node.js](https://nodejs.org/) y el CLI de Angular.

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/portafolio-juan-barrios.git
   cd portafolio-juan-barrios
   ```

2. **Instalar dependencias**:
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Ejecutar servidor de desarrollo**:
   ```bash
   npm start
   ```
   Abre tu navegador en `http://localhost:4200/`.
